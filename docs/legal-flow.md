# Legal/Consent Flow

สำหรับ MVP ให้เก็บ proof ขั้นต่ำ:

```txt
- contract version
- customer consent version
- acceptedAt
- IP address
- userAgent
- payment proof
- action approval log
```

ก่อนทำ action กับเครื่อง:

```txt
1. มีสัญญาที่ระบุเงื่อนไข
2. ลูกค้ายินยอมเงื่อนไขจัดการอุปกรณ์
3. มีการแจ้งเตือนก่อน
4. มีระยะผ่อนผัน
5. มี human approval
6. มี audit log
```

ห้ามใช้ระบบนี้เพื่อข่มขู่ ประจาน หรือควบคุมเครื่องที่เจ้าของไม่ยินยอม
