# Pilot to Production Plan

## Pilot 1 ร้าน
- ใช้ manual PromptPay
- ใช้ iCloud Custody Tracking ถ้ามีเคส iPhone เดิม
- ยังไม่เปิด MDM command แรง
- ใช้ manual review ทุก payment
- เก็บ feedback UI ร้าน

## Pilot 3-5 ร้าน
- เปิด Customer Portal เต็ม
- เปิด Risk Score + Collection
- เปิด Template Center ให้ร้านแก้ข้อความ
- ต่อ storage จริง
- เริ่มต่อ LINE notification

## Production
- ต่อ payment gateway/slip verification
- ต่อ Android Management API จริง
- ทำ Apple MDM/ADE เฉพาะร้าน/เครื่องที่พร้อม
- เปิด platform billing auto
- เปิด health score และ suspension policy
- ทำ backup/monitoring/security audit
