# ขอบเขตที่ต้องไม่หลุดสำหรับ Lease-to-own MDM

ใช้ได้เมื่อ:

```txt
- ร้าน/บริษัทถือกรรมสิทธิ์เครื่องก่อนส่งมอบ
- เครื่อง enroll ก่อนส่งมอบ
- ลูกค้าเซ็น lease-to-own + MDM consent + privacy notice
- policy ถูกใช้เพื่อ asset protection และ security baseline
- มีขั้นตอนแจ้งเตือน/ระยะผ่อนผัน/approval ก่อน limited mode
- จ่ายครบแล้ว release และโอนกรรมสิทธิ์
```

ห้ามทำ:

```txt
- แอบติดตั้ง MDM หลังขายเครื่องไปแล้ว
- ซ่อน profile
- เปิดกล้อง/ไมค์/อ่านข้อความ/ดูรูป
- track location ตลอดเวลาโดยไม่จำเป็น
- wipe เพราะค้างงวดเล็กน้อย
- lock ทันทีโดยไม่มี notice/review
- ไม่ปลด MDM หลังจ่ายครบ
```

คำที่ควรใช้ในระบบ:

```txt
Limited mode
Asset protection
Lease-to-own ownership retention
Release after paid-off
Ownership transfer
```

คำที่ไม่ควรใช้ขายหน้าบ้าน:

```txt
ล็อกเครื่องลูกหนี้
ค้างงวดปิดเครื่องทันที
ตามเครื่องได้ตลอด
ล้างเครื่องได้ทุกเมื่อ
```

พูดให้ธุรกิจดูเป็นธุรกิจ ไม่ใช่ร้านไสยเวทมือถือใส่ dashboard สวย ๆ
