# Operator Manual

## งานประจำวัน

1. เปิด Dashboard ดูยอดค้างและ action รออนุมัติ
2. ตรวจรายการชำระที่เป็น `VERIFYING` หรือ `PENDING`
3. กด confirm เฉพาะรายการที่ตรวจหลักฐานแล้ว
4. รัน Overdue Check หรือให้ cron รันอัตโนมัติ
5. ตรวจ Device Actions ก่อน approve ทุกครั้ง
6. Export CSV เก็บไว้ถ้าต้องการทำบัญชีแยก

## Flow ร้านมือถือ

### เพิ่มเครื่องเข้าระบบ

ไปที่เมนู `เครื่อง` แล้วกรอก Brand, Model, Platform, IMEI, Serial, Storage, Color

### เพิ่มลูกค้า

ไปที่เมนู `ลูกค้า` แล้วกรอกชื่อ เบอร์ ที่อยู่ และ risk score

### สร้างสัญญา

ไปที่เมนู `สัญญา` เลือกลูกค้า เลือกเครื่องในสต็อก กรอกราคา เงินดาวน์ ดอก/ค่าธรรมเนียม จำนวนงวด และวันครบกำหนดงวดแรก

### เซ็นสัญญา

กด `Sign` เพื่อเปลี่ยนสถานะจาก `DRAFT` เป็น `ACTIVE` และสร้าง consent records

### พิมพ์สัญญา

กด `Print` ในตารางสัญญา แล้วใช้ browser พิมพ์หรือบันทึกเป็น PDF

### ลูกค้าแจ้งชำระ

ลูกค้าเข้า Customer Portal ด้วยเลขสัญญาและเบอร์โทร เลือกงวด แล้วแจ้งชำระพร้อม link หลักฐาน

### ปิดสัญญา

เมื่อทุกงวดจ่ายครบ ระบบเปลี่ยนสัญญาเป็น `PAID_OFF` และสร้าง `REQUEST_RELEASE` ให้แอดมินอนุมัติปลดการจัดการเครื่อง
